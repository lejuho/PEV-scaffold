# PEV MCP

Local stdio MCP server for the PEV dashboard/Hermes/tmux control layer.

Tools:

- `pev_projects`, `pev_project`
- `pev_tail`
- `pev_command`
- `pev_flow`
- `pev_enter`
- `pev_say`
- `pev_create_done`
- `pev_tmux_sessions`

Commands are constrained to the same allowlist used by the dashboard.
